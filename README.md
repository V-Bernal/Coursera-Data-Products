# Coursera-Data Products
R shinny app
================
This Shiny App is designed for visualizing clusters in the mtcars dataset. The **mtcars** data displays several features of car design and performance for 32 vehicles.Source: the 1974 Motor Trend US magazine.

To explore the data you can select three features (x, y axis and size of the symbol). You will observe the clustering (according to your variables of choice) displyed with colors. The center of the cluster is shown in black.
